# Traffic_Light_Controller_VerilogHDL
Verilog HDL based Traffic Light Controller designed, simulated, and verified in Vivado for FPGA and digital design applications.

## Introduction

This project implements a Traffic Light Controller using Verilog HDL. The controller follows a predefined sequence of traffic signals and demonstrates the application of Finite State Machine (FSM) concepts in digital design. The project was designed, simulated, and synthesized using AMD Vivado.

## Project Overview

The main objective of this project is to design a traffic signal control system that manages the transition between Red, Yellow, and Green lights in a systematic manner. The design was verified using a testbench and simulation waveforms.
